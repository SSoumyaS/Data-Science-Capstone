Prediction Model

Find word count, separate words, lower case
input_count <- str_count(input, boundary(“word”))
input_words <- unlist(str_split(input, boundary(“word”)))
input_words <- tolower(input_words)

Look for the matching functions
out <- ifelse(input_count == 0, “Please input a phrase”,
ifelse(input_count == 3, quadgram(input_words),
ifelse(input_count == 2, trigram(input_words),  bigram(input_words))))

Function Example

(Using the Quadgram based function)

num <- length(input_words)
filter(quad_words,
word1==input_words[num-2],
word2==input_words[num-1],  word3==input_words[num]) %>%
top_n(1, n) %>%
filter(row_number() == 1L) %>%
select(num_range(“word”, 4)) %>%
as.character() -> out
